Title: Titanic Survival Analysis: Data Visualization in Tableau
Description:
This project contains a set of Tableau visualizations and analyses performed on the Titanic dataset. 
The aim was to explore various passenger attributes (like gender, age, class, and fare) against the survival outcome. 
Key visualizations include bar charts for gender distribution and survival counts, heatmaps for survival rate by class/gender, box plots for age distribution, and scatter plots for fare vs. age. 
The work follows a guided hands-on exercise to practice fundamental data visualization techniques.



Title: Coffee Chain Performance Analysis: Tableau Dashboarding
Description:
It doesTableau analyses of the Coffee Chain dataset, focusing on identifying trends in Profit, Sales, and Marketing across various product types, states, and markets.
The project includes step-by-step instructions and resulting visualizations for tasks like creating heatmaps (Product Type, State, Profit), box plots (Sales distribution), bar charts (Product/Profit breakdown), and addressing specific business questions through calculated metrics and filters (e.g., contribution of tea to profit, average marketing by area code, top/lowest profit states).
